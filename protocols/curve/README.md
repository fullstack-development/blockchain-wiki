# Curve

**Автор:** [Павел Найданов](https://github.com/PavelNaydanov) 🕵️‍♂️

**Curve** — это децентрализованная обменник (DEX) на базе автоматизированного маркет-мейкера (AMM) для Ethereum и совместимых с EVM блокчейнов. По большей части предназначен для эффективного обмена стейблкоинами или обычными волатильными активами.
