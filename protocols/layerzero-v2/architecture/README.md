#### Ultra Light Node

**Ultra Light Node (ULN)** ‚Äî —ç—Ç–æ –±–∞–∑–æ–≤–∞—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ —Å–æ–æ–±—â–µ–Ω–∏–π (*MessageLib*), –≤–∫–ª—é—á–µ–Ω–Ω–∞—è –≤ –∫–∞–∂–¥–æ–µ —Ä–∞–∑–≤–µ—Ä—Ç—ã–≤–∞–Ω–∏–µ LayerZero. *ULN* –ø–æ–∑–≤–æ–ª—è–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–æ 254 *DVNs* —á–µ—Ä–µ–∑ –Ω–∞—Å—Ç—Ä–∞–∏–≤–∞–µ–º—É—é –¥–≤—É—Ö—É—Ä–æ–≤–Ω–µ–≤—É—é —Å–∏—Å—Ç–µ–º—É –∫–≤–æ—Ä—É–º–∞.

*ULN* —Ä–µ–∞–ª–∏–∑—É–µ—Ç –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –Ω–∞–±–æ—Ä —Ñ—É–Ω–∫—Ü–∏–π, –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã—Ö –¥–ª—è –ª—é–±–æ–≥–æ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –≤–µ—Ä–∏—Ñ–∏–∫–∞—Ü–∏–∏, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –µ–≥–æ —Å–æ–≤–º–µ—Å—Ç–∏–º—ã–º —Å –ª—é–±—ã–º–∏ –±–ª–æ–∫—á–µ–π–Ω–∞–º–∏. –ö–∞–∂–¥—ã–π *Security Stack* OApp, –∏—Å–ø–æ–ª—å–∑—É—é—â–∏–π *ULN*, –≤–∫–ª—é—á–∞–µ—Ç:

- –û–±—è–∑–∞—Ç–µ–ª—å–Ω—ã–µ *DVNs*.
- –û–ø—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–µ *DVNs* –∏ –ø–æ—Ä–æ–≥ –¥–ª—è –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω—ã—Ö –≤–µ—Ä–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–≤ (*OptionalThreshold*).
