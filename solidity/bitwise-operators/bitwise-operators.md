    n << 3 == n * 8   |   n >> 1 == n / 8
```

_–í–∞–∂–Ω–æ!_ –î–µ–ª–µ–Ω–∏–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Å –æ–∫—Ä—É–≥–ª–µ–Ω–∏–µ–º –≤ –º–µ–Ω—å—à—É—é —Å—Ç–æ—Ä–æ–Ω—É, –Ω–∞–ø—Ä–∏–º–µ—Ä `9 >> 1 = 4`.

_–í–∞–∂–Ω–æ!_ –°–¥–≤–∏–≥ –≤–ø—Ä–∞–≤–æ —á–∏—Å–ª–∞ (-1) –≤—Å–µ–≥–¥–∞ –±—É–¥–µ—Ç –¥–∞–≤–∞—Ç—å (-1), –≤—ã–≥–ª—è–¥–µ—Ç—å —ç—Ç–æ –±—É–¥–µ—Ç —Ç–∞–∫:

```

_–ü–æ—Ç–æ–º—É —á—Ç–æ –∫–∞–∫ –º—ã –ø–æ–º–Ω–∏–º –ø—Ä–∏ —Å–¥–≤–∏–≥–µ —á–∏—Å–ª–∞ –≤–ø—Ä–∞–≤–æ —Å—Ç–∞—Ä—à–∏–π –±–∏—Ç –∑–∞–ø–æ–ª–Ω—è–µ—Ç—Å—è –±–∏—Ç–æ–º –∑–Ω–∞–∫–∞, —É –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ —á–∏—Å–ª–∞ —ç—Ç–æ 1._
